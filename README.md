# 🛒 Sportify – E-Commerce Web Application

Sportify is a full-stack sports-based e-commerce web application built using Django, MySQL, and Bootstrap. It allows users to browse and shop sports products across multiple categories like Football, Cricket, and Fitness.

## 🔧 Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** MySQL
- **Others:** Django ORM, Templates, Session Management

## ✅ Features

- User Registration & Login
- Category-wise Product Browsing
- Product Detail View
- Add to Cart / Wishlist Functionality
- Admin Panel for Product Management
- Responsive UI Design
